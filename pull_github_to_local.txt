#### Connecter la clé SSH publique au compte GitHub (Dans les paramètres de GitHub)





#### Ajouter la clé SSH à l'agent [L'agent SSH est un petit programme qui stocke en mémoire ta clé privée SSH pendant ta session. Cela évite de devoir entrer ton mot de passe ou recharger ta clé à chaque commande Git (comme git clone, git push, etc.)]

## Démarrer l’agent SSH 
eval "$(ssh-agent -s)"

## Ajout de la clé
ssh-add "/c/Users/EugèneDettaa/OneDrive - Statlog consulting/07_Projet_Git/.ssh/id_ed25519"

## Tester la connexion à GitHub
ssh -T git@github.com
